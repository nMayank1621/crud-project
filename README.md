# Django CRUD Project

A simple CRUD (Create, Read, Update, Delete) web app built with Django.

## Features

* Create, update, delete records
* List view with templates
* Clean UI

## Tech Stack

* Python, Django
* SQLite
* HTML, CSS

## Setup

```bash
git clone https://github.com/nMayank1621/crud-project.git
cd crud-project
python -m venv venv
venv\Scripts\activate
pip install django
python manage.py migrate
python manage.py runserver
```

Open: http://127.0.0.1:8000/
